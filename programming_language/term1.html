基本データ型

型 8ビット符号つき整数
型 16ビット符号つき整数
型 32ビット符号つき整数
型 64ビット符号つき整数
型 32ビット浮動小数点数
型 64ビット浮動小数点数
型 16ビットUnicode文字
型 1ビット真偽値

※String型は基本データ型ではない！
（基本データ型じゃないので最初は大文字と覚えたら良い？）
あと基本データ型じゃないので==での比較ができません(上の基本データ型は全て==で比較できます)

printf文の書式指定

% 整数を10進数で出力
% 実数を浮動小数点数で出力
% 実数を指数形式で出力
% 文字を出力
% 文字列を出力

printf文の幅指定つき書式指定

% 8桁の10進数で出力
% 左寄せで8桁の10進数で出力
% 浮動小数点数を幅10桁で出力、小数点以下は5桁で出力
% 整数部は必要幅、小数点以下は5桁で出力

文字列どうしを比較するメソッド(等しいか否かを判定)(戻り値:boolean)
文字列の指定されたインデックスにある文字を取得するメソッド
